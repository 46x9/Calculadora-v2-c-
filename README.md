# 🧮 Calculadora Versão 2 (C++)

Uma calculadora simples e funcional desenvolvida em C++ que realiza operações básicas de aritmética em um laço contínuo até que o usuário decida sair.

## 🚀 Funcionalidades

O programa suporta as seguintes operações:
* **Soma** (`+`)
* **Subtração** (`-`)
* **Multiplicação** (`*`)
* **Divisão** (`/`)
* **Sair**: Digite `sair` para encerrar o programa.

## 🛠️ Como executar

Para compilar e rodar este projeto no seu computador, você precisará de um compilador de C++ (como o `g++`).

1. Abra o terminal na pasta do projeto.
2. Compile o código:
   ```bash
   g++ std.cpp -o calculadora
